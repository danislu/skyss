





# Skyss API

**_Work in progress_**

Library intended to be used for a Pebble app when I get around to it.

## Installation

* `git clone https://github.com/danislu/skyss.git`
* `cd skyss`
* `npm install`

## Usage

Build: `npm run build`

Test: `npm run test`
